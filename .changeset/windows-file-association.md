Windows now registers the `.stitches` file association at launch — double-clicking a `.stitches` file in Explorer opens it directly in Stitches. No admin rights required (written to `HKCU`). Matches existing macOS/iOS/Android behaviour.

Drag and drop a `.stitches` file onto the home screen to open it on macOS, Windows, and Linux.
