In stitch mode with page mode enabled, add an "All / Page" toggle to the Colours panel. When "Page" is selected, only threads that have stitches on the current page are shown, with counts scoped to that page. Uses the composite (flattened) cache for full stitches so only the topmost visible colour per cell is counted. The toggle is hidden in view mode and when page mode is not active.
